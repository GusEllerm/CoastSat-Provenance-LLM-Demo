# Test Files Directory

This directory contains Stencila documents for testing your local build of the CLI.

## Directory Structure

- `inputs/` - Source documents in various formats for testing conversions
- `outputs/` - Generated output files from CLI operations
- `examples/` - Example Stencila documents you can experiment with

## Quick Start

### Test Document Conversion

```bash
# Convert JSON to YAML
cd stencila
cargo run --bin stencila -- convert ../test-files/inputs/article.json ../test-files/outputs/article.yaml

# Convert YAML to JSON
cargo run --bin stencila -- convert ../test-files/inputs/article.yaml ../test-files/outputs/article.json
```

### Test Document Execution

```bash
cd stencila
cargo run --bin stencila -- execute ../test-files/examples/document-with-code.smd
```

### Test Document Rendering

```bash
cd stencila
cargo run --bin stencila -- render ../test-files/examples/article.smd
```

## Example Files

Check the `examples/` directory for sample Stencila documents you can use for testing.

## Tips

- Use `inputs/` for source files you want to test conversions with
- Use `outputs/` for files generated by CLI commands (this directory is gitignored)
- Use `examples/` for complete example documents
